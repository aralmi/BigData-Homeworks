# Big Data Homeworks

Домашние задания по обработке больших данных с использованием **Apache Spark** и **Python**.

### Домашнее задание №1. Spark RDD API
[Distance Calculations in Moscow](https://github.com/aralmi/BigData-Homeworks/blob/main/hw1)

Расчет расстояний между заведениями общепита используя формулу гаверсинуса. Работа с RDD, функции map, filter, cartesian.


### Домашнее задание №2. Spark DataFrame API
[Citibike Analysis with Geospatial Mapping](https://github.com/aralmi/BigData-Homeworks/blob/main/hw2)

Анализ системы велошеринга Citibike в Нью-Йорке. Сопоставление координат станций с таксономическими зонами и визуализация через интерактивные Choropleth карты.


### Домашнее задание №3. Spark Streaming
[Telegram Producer & Spark Streaming Consumer](https://github.com/aralmi/BigData-Homeworks/blob/main/hw3)

Разработка потокового приложения с использованием Spark Streaming. Интеграция с Telegram API для получения данных в реальном времени и их обработки.


### Домашнее задание №4. Spark MLlib
[Recommendation System (Collaborative Filtering + ALS)](https://github.com/aralmi/BigData-Homeworks/blob/main/hw4)

Реализация и сравнение трёх методов предсказания рейтингов фильмов: Базовая модель → Коллаборативная фильтрация → Факторизация матриц (ALS).
